Browser: set CLOUDINIT_BROWSER_ONLY=1 and open this app’s port (see console). Desktop: install pywebview and run python app.py (embedded window). Seed path override: CLOUDINIT_SEED_DIR. Outputs: output/user-data, output/meta-data. Backup: backup/preferences.json (removed when you clear preferences).

When running in Docker, use Download NoCloud ISO (cidata) in the Output section after Generate. Or, from a seed folder on the host, create an ISO with:
genisoimage -output example.iso -volid cidata -joliet -r output/
You can then use the ISO file to boot a virtual machine.
virt-install --name=ubuntu-server --memory=1024 --disk path=example.iso,device=cdrom --os-variant=ubuntu22.04 --network=bridge=br0 --graphics=none --console=pty,target_type=serial --boot=network,cdrom
